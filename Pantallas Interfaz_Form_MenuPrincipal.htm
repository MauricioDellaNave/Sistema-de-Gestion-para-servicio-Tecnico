  • Sesión 

Permite escoger entre dos opciones Salir del sistema o Cerrar la sesión actual.

         Cerrar Sesión

            Permite cerrar la sesión actual del sistema permitiendo realizar el loguin nuevamente.

         Salir

            Permite cerrar el sistema

  • Administrar

        Clientes

           Despliega un formulario de ABM de clientes

        Reparaciones

           Despliega un formulario de ABM de Reparaciones

        Equipos

           Despliega un formulario de ABM de Equipos

        Tareas

           Despliega un formulario de ABM de Tareas

        Repuestos

           Despliega un formulario de ABM de Repuestos

  • Seguridad

Despliega opciones de Seguridad del Sistema

         Resguardo y Restauracion

            Permite realizar copias de Resguardo de la base de Datos y restaurar la base de Datos en caso de ser necesario.

         Usuarios

            Despliega un formulario de ABM de Usuarios

         Familia y Patente

            Despliega un formulario de Modificaciones de Familias y Patentes

         Bitacora

            Despliega un formulario de visualización de bitacoras del sistema 

  • Reparaciones

Despliega opciones de Ingresos y reparaciones de equipos 

        Ingresar Equipos a reparación

          Despliega un formulario de ingreso de equipos a Servicio Tecnico

        Realizar Reparaciones

          Despliega un formulario que permite realizar Reparaciones y Presupuestos

 

  • Consultar

Despliega opciones realizar consultas sobre equipos en reparacion y equipos por clientes

        Historial Equipo

           Despliega un formulario donde debo ingresar el numero de serie del equipo a buscar.

        Historial Cliente

           Despliega un formulario donde debo seleccionar el cliente a consultar.
